Enlaces

Aplicación principal, Aplicación de prueba y memoria del proyecto

El objetivo de este proyecto de fin de máster ha sido desarrollar una aplicación web capaz de escanear servidores externos en busca de una o varias vulnerabilidades. La vulnerabilidad escogida ha sido SQL Injection. El proyecto consta de dos aplicaciones web: la principal, responsable de realizar el análisis y una secundaria, vulnerable a ataques de inyección SQL para probar el correcto funcionamiento de la aplicación principal.

Para conseguir realizar el análisis he utilizado una aplicación externa, llamada SQLmap, con la que se puede realizar un análisis completo de la vulnerabilidad con solo introducirle los parámetros de las consultas. Debido a mi preferencia a trabajar en Debian en el desarrollo del servidor sólo he tenido que introducir la aplicación dentro del servidor y ejecutar los comandos adecuados con PHP para crear una interfaz desde la que se puedan conseguir los objetivos marcados.

Con respecto a la aplicación de pruebas, solamente tengo que añadir que es muy sencilla en todos los aspectos y solamente la utilizo como pruebas de la aplicación principal.

Para saber más acerca de cómo utilizar la aplicación basta con acceder a la misma ya que dentro tiene un apartado en donde se explica cómo utilizarla. Para más información acerca del funcionamiento interno se puede consultar la memoria del proyecto, que se puede encontrar en el enlace correspondiente en la cabecera de esta página.
